# Eventhouse and KQL Model

## Purpose

Documents real-time tables, KQL queries, and playback export logic.

## Key Tables

- `rt_bus_vehicle_positions`
- `rt_bus_trip_updates`
- `rt_service_status`
- `rt_service_alerts`

## KQL Usage

KQL is used for:

- Playback frame generation
- Data freshness checks
- Data quality checks
- Service alert analytics
- Operational KPI queries
