# Fabric Pipeline Design

## Pipeline Name

`pl_stm_scheduled_collection_mvp`

## Purpose

Scheduled collection pipeline for STM transit data.

## Main Activities

1. Bus positions ingestion
2. Trip updates ingestion
3. Metro service status ingestion
4. Service alerts ingestion
5. Dashboard freshness status update

## Schedule

The animation dataset was captured using 5-minute collection intervals.
